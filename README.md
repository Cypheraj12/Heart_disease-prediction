# Heart Disease Predictor

## Overview

Heart Disease Predictor is a machine learning web application built with Streamlit that estimates the likelihood of heart disease based on patient health information. The application uses a Logistic Regression model trained on the UCI Heart Disease dataset and provides real-time predictions through an interactive interface.

The goal of this project is to demonstrate how machine learning can be applied to healthcare data to support early risk assessment and awareness.

---

## Features

* Interactive web interface built with Streamlit
* Real-time heart disease risk prediction
* Logistic Regression classification model
* Data preprocessing and missing value handling
* Feature engineering for improved model performance
* Probability-based prediction results
* Visual risk indicators and confidence scores
* Responsive and user-friendly design

---

## Dataset

This project uses the UCI Heart Disease dataset, which contains patient demographic and clinical information such as:

* Age
* Sex
* Chest pain type
* Resting blood pressure
* Cholesterol level
* Fasting blood sugar
* ECG results
* Maximum heart rate achieved
* Exercise-induced angina
* ST depression (Oldpeak)
* Number of major vessels
* Thalassemia test results

The target variable indicates the presence or absence of heart disease.

---

## Machine Learning Pipeline

The model training process includes:

1. Data cleaning and preprocessing
2. Missing value treatment
3. Feature engineering
4. One-hot encoding of categorical variables
5. Feature scaling using StandardScaler
6. Model training using Logistic Regression
7. Performance evaluation using accuracy score

## Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/heart-disease-predictor.git
cd heart-disease-predictor
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```

The application will start on your local machine and open in your browser.

## How It Works

1. Enter patient information using the input form.
2. Click the prediction button.
3. The application processes the input data.
4. The trained model evaluates the information.
5. A prediction is generated along with probability scores indicating the estimated risk level.
